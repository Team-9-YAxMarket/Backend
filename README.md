# backend


## Стек технологий, использованных в проекте
- python 3.10
- fastapi
- fastapi-restful
- pydantic
- alembic
- asyncpg
- aiohttp
- uvicorn
- python-dotenv
- sqlalchemy
- PostgreSQL

## Линтеры
Весь код должен автоматически форматироваться `isort` и `black`.
Весь код должен проходить проверку `flake8`.
